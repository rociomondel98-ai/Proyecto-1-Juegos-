PROYECTO JUEGOS RETO - EQUIPO 1 S.L

Repositorio que contiene dos juegos desarrollados en Python por Equipo S.L después de que un cliente nos haya encargado desarrollar dos juegos de temática retro con las siguientes directrices: que los juegos sean intuitivos, seguros ante errores, altamente re-jugables y que estén pensados para captar y retener jugadores. 

Los juegos han sido desarrollados en Jupyter Notebook por las cinco integrantes de Equipo 1 implementando la metodología Scrum que nos ha permitido tener una coordinación efectiva entre organización y desarrollo.
